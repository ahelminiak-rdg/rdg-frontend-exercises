Learn Sass

Sass stands for "Syntactically awesome style sheets." It is an extension of CSS that allows developers to write more flexible CSS by using variables, mixins, conditional statements, and more.

Exercise 1: Style Guide

Using the HTML on this page and sass file structure that has already been set up for you, create a sass/css style guide based on the provided mockup. Start by styling the small elements of the page (i.e. typography, links, buttons, etc.) and gradually build up to larger components/regions. Try to utilize sass variables and mixins to help with uniformity and to reduce duplicate code.
