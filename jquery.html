Exercise 1: This

jQuery's "This" is selector refers to the current selected object in a loop or event. It allows you to select only one element in a function, even if there are other matching elements. (For example, clicking on an accordion element to expand only the one you clicked on, rather than all accordions on the page). In this exercise, you will write a simple jQuery function that only logs the text inside of the link that you click on.

Exercise 3: If/Else Statements

If/else statements (or conditional statements) are a way for you to perform different actions based on different conditions. If a condition is true (an element has a class, an element exists on the page, etc.), then perform one action, otherwise perform a different action. In the exercise below, write a function that checks if any of the elements have the class "highlighted" when clicking the button. If they do, log "TRUE" in the console and remove the highlighted class. If the highlighted class does not exist, then log "FALSE" in the console (HINT: you should have 2 FALSE and 1 TRUE logged in your console)
